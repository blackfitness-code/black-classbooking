# Product

## Register

product

## Users

Staff และ admin ของ Blackfitness ที่นั่งหน้าเคาน์เตอร์หรือในออฟฟิศ ใช้จอ desktop ตรวจสอบสมาชิกใหม่ ปรับแพ็คเกจ/วันหมดอายุ และดูข้อมูลที่ลูกค้ากรอกจาก LINE LIFF

## Product Purpose

Dashboard สำหรับ monitor สมาชิกที่สมัครและกรอกโปรไฟล์ครบ แจ้งเตือนแอดมินทันที และให้แก้ไขข้อมูล/สมาชิกภาพได้โดยไม่ต้องเปิด LIFF app

## Brand Personality

กระชับ เป็นมืออาชีพ อบอุ่น — พลังงานฟิตเนสผ่าน accent สี coral ไม่ใช่ UI ที่ตะโกน

## Anti-references

- Dashboard SaaS สีครีม/เทา generic
- Card grid ซ้ำ ๆ รอบตาราง
- Hero metrics (ตัวเลขใหญ่ + label เล็ก)
- Uppercase eyebrow ทุก section
- Modal เป็นทางเลือกแรกเมื่อ inline edit ทำได้

## Design Principles

1. **งานก่อนตกแต่ง** — แอดมินต้องเห็นสมาชิกใหม่และแก้ไขได้ใน ≤2 คลิก
2. **สแกนได้เร็ว** — ตารางหนาแน่น อ่านชื่อ/เบอร์/แพ็คเกจได้ในที่เดียว
3. **แจ้งเตือนชัด ไม่รบกวน** — กระดิ่ง + badge ไม่ใช่ panel ถาวร
4. **ภาษาไทยอ่านสบาย** — contrast ผ่าน AA, ไม่ใช้เทาจางบนพื้นขาว
5. **สอดคล้อง LIFF app** — Kanit + coral primary แต่โทน product (Restrained)

## Accessibility & Inclusion

WCAG 2.1 AA สำหรับ contrast และ focus states; รองรับ `prefers-reduced-motion`
